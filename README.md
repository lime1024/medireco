## アプリ概要
「メディレコ」というサービスは、「医療費控除の記録の面倒さ」を解決したい。「医療費控除を申請したいけど記録が面倒だと思っている人(自分)」向けの、「記録サービス」です。ユーザーは「家族全員の医療費、交通費の記録」ができ、他アプリとは違って、「PC閲覧とエクセル出力」が備わっている事が特徴です。

## 医療費控除とは
10万円以上の医療費を年間で支払ったときに、納めた税金の一部が戻ってくるというもので確定申告が必要です。
医療費控除は**家族の分も合算して申告が可能**で、病院での**治療費**だけで無く、**医薬品費**、通院に必要な**交通費**も医療費控除の対象となります。

詳細 : [No.1120 医療費を支払ったとき（医療費控除）｜国税庁](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1120.htm)

## メディレコの特徴
医療費を記録すると、国税庁指定の「医療費集計フォーム」がダウンロードできるようになります。[確定申告書等作成コーナー](https://www.keisan.nta.go.jp/kyoutu/ky/sm/top#bsctrl)の医療費控除の入力画面で読み込みを行い、反映することで簡単に医療費控除の申請を行うことができます。
